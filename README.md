A modern IRC client built with web technologies and Node.js for OS X, Linux and Windows.

# Features

* Themeable with CSS
* Scriptable with Javascript
* Keyboard shortcuts
* Minimal
* Open source
* Built with web technologies for extensibility (Node.js and Electron)
